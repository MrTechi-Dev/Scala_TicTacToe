# Tic tac toe

Simple Scala 3 

## Project Setup

1. Clone this repository to any destination path you want
    
2. Open terminal and navigate to this directory:
   ```bash
    $ Your path in your local machine
   ```
4. Compile project:
   ```bash
   $ sbt compile
   ```
5. Run tests:
   ```bash
   $ sbt test
   ```
6. Run tic-tac-toe game:
   ```bash
   $ sbt run
   ```
